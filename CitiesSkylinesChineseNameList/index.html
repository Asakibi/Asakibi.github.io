Cities: Skylines 本地化嘗試

該文章尚未編輯完成，請以最終版本為準。

　　最近遊玩 Cities: Skylines 简体中文版，發現遊戲內的翻譯問題較多，本地化工作做得並不到位。如出現以下問題：

  1. 姓名顛倒：英文版採用名（First Name）和姓（Last Name）的方法組合人名，而简体中文版雖沿用了這一方式，但將英文名設置為中文姓，將英文姓設置為中文名。
  2. 罕見名字：如有「奶奶」「大妈」「姨」「娘娘」等十分奇怪的名字。
  3. 名字男女共用：男女共用一套名字列表，這使得男性也會有「菲菲」「奶奶」「大妈」等名字，不符合實際情況。
  4. 非常規路名：以 Last Name/Residence Name/District Name + 路/大街/干道… 命名。如會出現「奶奶 干道」「山毛榉 街」等含不明原因空格的不符合中文表達習慣的道路名稱。
  5. 錯誤翻譯：如收費站的 charge 應譯作「收費」而非「充電」。

　　除以上問題以外，住宅和商業建築命名方式也有待改善。於是萌生了製作一款 mod 以使遊戲的翻譯符合中國人語言使用習慣的念頭。
　　在 Steam 創意工坊上搜尋到了 Steam 創意工坊 - Custom Names Lists mod。據作者的描述，通過創建 .namelist 和 .blacklist 文件（事實上是 xml 文件）作為白名單（用於添加翻譯）和黑名單（用於屏蔽原生翻譯），可以修改遊戲中居民和建築等的備選名稱列表。

該文章將在編輯完成後推出 简体中文版。

查看遊戲的原生翻譯

　　遊戲的語言文件在 {source}\Files\Locale 下。遊戲原生的翻譯可藉由 Steam 社區 - Localization Tool 查看。Custom Names Lists mod 也自帶導出功能。點擊遊戲中的 export 按鈕，桌面上會生成一個名為 complete.namelist 的文件。透過該文件可以查詢遊戲原生翻譯。

問題的解決

1. 姓名问题

　　原生的語言文件採用 {0} 代表一個字符串中可供插入的位置，以組合字符串。如 NAME_MALE_FIRST（男性名字）的 1 個字符串是 李，NAME_MALE_LAST（男性姓氏）的第 2 個字符串是 {0}佳佳，兩者組合得姓名 李佳佳。
　　那麼要更正遊戲的中文姓名顛倒問題，應將 Last Name 設置為 姓{0}，將 First Name 設置為 名，經組合得 姓名。

2. 道路名問題

　　遊戲採用 Last Name/Residence Name/District Name + 道路類型 方式命名道路，而通配符 {0} 被放置在道路類型名稱的字符串中。於是，要使路名有其獨特名字體系，需要刪除該通配符，修改為諸如「建设路」「人民大道」的字符串。
